What we use

Asset creation

Krita — Used for creating game assets, including new HUD elements and other visual designs.

FL Studio — Used for composing and producing the game’s background music and sound effects.

Development

Unity Engine — The core engine used for building, testing, and running the game.

JetBrains Rider — Used as an IDE for C# development within Unity.

Github — Used for version control, issue tracking, and pull request reviews.

Organization & Communication

Google Sheets — Used for task management, tracking sprint progress and assigning issues across the team.

Google Forms — Used to collect responses from public testers during the larger testing rounds.

Discord — Used for day-to-day team coordination, quick questions, and hotfix discussions.

Slack — Used for communication with the professor and gathering her feedback.

Zoom — Used for weekly meetings.
